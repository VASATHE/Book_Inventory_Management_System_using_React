# Book Inventory Management System

A full stack web application built using:

- React (Frontend)
- Spring Boot (Backend)
- MySQL (Database)

## Features

- View all books in table
- Add, update, delete books
- REST API integration
- Form validation
- Responsive UI
- MySQL storage

## Tech Stack

Frontend:
- React
- Axios

Backend:
- Spring Boot
- Spring Data JPA
- MySQL

## Project Structure

book-inventory-system
│
├── backend
└── frontend

## How to Run

### Backend

1. Create database:

```sql
CREATE DATABASE Book_Inventory;

2. Update MySQL credentials in application.properties

3. Run:
    mvn spring-boot:run

Backend runs on:

http://localhost:8083

Frontend

cd frontend
npm install
npm start


Frontend runs on:

http://localhost:3000
